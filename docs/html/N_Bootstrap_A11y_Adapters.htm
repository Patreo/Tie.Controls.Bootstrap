Bootstrap.A11y.Adapters Namespace
The Bootstrap.A11y.Adapters namespace contains WebControlAdapters for applying basic Bootstrap styling to select ASP.NET user controls.
Classes
  ClassDescription
ButtonAdapter
Adapter to apply Bootstrap classes to a base ASP.NET Button.
GridViewAdapter

InputAdapter
Adapter to apply Bootstrap classes to a base ASP.NET input (e.g. TextBox, DropDownList, or ListBox).
